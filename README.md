# 2018년 서울시 앱 공모전 출품작(check-mate)

## Mash-Up 5기 프로젝트팀 체크메이트 : 위치 기반 급여 자동 계산 앱

### 배경
 - "내가 이번 달 얼마나 일했지? 총 얼마를 받아야 할까?"
 - "사장님이 계산하신 시급이랑 내가 계산한 시급이 다른데 어떻게 말씀드릴까?"  
 - 위와 같은 고민으로부터 아이디어를 실현 시켰습니다.
